# KPI Infrastructure Automation Lab

Це проєкт автоматизації розгортання інфраструктури в середовищі KVM з використанням Terraform та Ansible.

## Структура проєкту
- `terraform/` — описи ресурсів для віртуальних машин (VM).
- `ansible/` — ролі та плейбуки для конфігурації ПЗ (Java, Nginx, MySQL).
- `app.jar` — ваш готовий Java-застосунок.

## Вимоги
- Встановлені `terraform` та `ansible`.
- Налаштований `libvirt` (KVM/QEMU) з доступом до `/dev/kvm`.
- Наявність SSH-ключа. 

## Інструкція по запуску

### 1. Підготовка
Переконайтеся, що файл вашого застосунку знаходиться в корені проєкту під назвою `app.jar`.

### 2. Розгортання інфраструктури (Terraform)
Перейдіть у директорію Terraform, ініціалізуйте провайдери та запустіть розгортання:
```bash
cd terraform
terraform init
terraform apply -auto-approve

Після завершення Terraform автоматично створить дві ВМ та поверне їхні IP-адреси.
